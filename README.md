Olimob Test Case

SPA на React с имитацией авторизации

Стек:
React + TS
Zustand + persist(для работы с локальным хранилищем)
FSD
SCSS

Что успел реализовать за 4 часа:

Страница логина с валидацией
Имитация загрузки при логине
Сохранение пользователя в localStorage
Приватные маршруты
Страница профиля
Logout с очисткой состояния
404 страница с возм перехода на главную

Что хотелось бы доделать:

Добавил бы лоадер при загрузках
Произвел бы небольшой рефакторинг кодовой базы (логику бы разнес по хукам, 
добавил бы имитацию лоадера при переходе с 404 на главную)
Покрыл бы тестами
В идеале бы написать простенький сервер на ноде и избавиться от имитации запросов


P.S. Старался придерживаться FSD архитектуры. До этого момента не имел опыта работы с zustand, 
был приятно удивлен отсутствию бойлерплэйта как в редаксе, полезным фишкам вроде persist и простоте использования.
